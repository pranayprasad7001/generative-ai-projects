````markdown
# Q&A Chatbot with Groq ⚡

A lightweight Question & Answer chatbot built with LangChain, Groq, and Streamlit. The application leverages LangChain Prompt Templates to structure prompts before sending them to Groq-hosted open-source language models.

## 🚀 Live Demo

🔗 https://question-answer-chatbot-groq.streamlit.app

## ✨ Features

- 🤖 Chat with multiple Groq-hosted LLMs
- 📝 Prompt engineering using LangChain Prompt Templates
- 🎛️ Adjustable temperature and max tokens
- 🧠 View model reasoning (for supported reasoning models)
- 🔑 Secure API key input
- ⚡ Fast and interactive Streamlit interface

## 🛠️ Tech Stack

- Python
- LangChain
- Groq API
- Streamlit
- Python Dotenv

```bash

cd generative-ai-projects/langchain-projects/qa-chatbot

pip install -r requirements.txt
````
